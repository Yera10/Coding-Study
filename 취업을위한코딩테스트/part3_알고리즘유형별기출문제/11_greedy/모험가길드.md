# 모험가 길드

### Input
- N : 첫째 줄에 주어지는 모험가의 수 
- 1 <= N <= 100,000
- 각 모험가의 공포도 : 둘째 줄에 주어지는 N이하의 자연수 값들, 공백으로 구분됨

### Output
- 여행을 떠날 수 있는 그룹 수의 최대값

### Rule
- 공포도가 X인 모험가는 반드시 X명 이상으로 구성한 모험가 그룹에 참여해야 여행을 떠날 수 있음
- 최대로 만들 수 있는 모험감의 그룹?

### Example
| 입력 | 출력 |
| ---- | --- |
| 5 <br> 2 3 1 2 2 | 2 |
